# Git Commands Cheat Sheet

## Configuration
- `git config --global user.name "Your Name"`
- `git config --global user.email "your@email.com"`

## Initialize Repository
- `git init`

## Clone Repository
- `git clone <repo-url>`

## Check Status
- `git status`

## Add Files
- `git add <file>`
- `git add .`

## Commit Changes
- `git commit -m "message"`

## Push Changes
- `git push`
- `git push origin <branch>`

## Pull Changes
- `git pull`
- `git pull origin <branch>`

## Branching
- `git branch`
- `git branch <branch-name>`
- `git checkout <branch-name>`
- `git checkout -b <branch-name>`

## Merging
- `git merge <branch-name>`

## Stashing
- `git stash`
- `git stash apply`

## Logs
- `git log`
- `git log --oneline`

## Remote
- `git remote -v`
- `git remote add origin <url>`

## Undo & Reset
- `git restore <file>`
- `git reset --hard`
- `git reset --soft`

## Tags
- `git tag`
- `git tag <tag-name>`
- `git push origin <tag-name>`
